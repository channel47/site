# Channel 47 Site

Astro 5 site deploying to channel47.dev via Vercel.

## Commands

```bash
npm run dev      # localhost:4321
npm run build    # Production build
```

## Environment

```bash
BEEHIIV_API_KEY=
BEEHIIV_PUBLICATION_ID=
```

Static output with one serverless endpoint (`api/subscribe.ts`).
